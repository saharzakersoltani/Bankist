#  Bankist: A Minimalist Online Banking App  

Bankist is a sleek, fictional online banking interface designed to simulate core banking functionalities like transfers, loans, and account management—with a focus on **clean UI/UX** and **JavaScript-driven interactions**.  

---

###  Key Features  
- **User Login**: Secure authentication with usernames and PINs.  
- **Money Transfers**: Send funds to other Bankist users instantly.  
- **Loan Requests**: Simulated loan approval based on account rules.  
- **Account Summary**: Clean display of transactions, balance, and interest.  
- **Auto Logout**: Inactive sessions expire after 5 minutes for security.  
- **Sortable Transactions**: View transactions in chronological or amount-based order.  

---

##  Technologies Used  
- **HTML5** / **CSS3** 
- **JavaScript** (DOM manipulation, array methods, timers)
